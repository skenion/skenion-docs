# Skenion Manual 0.43

The 0.43 Manual track covers the Skenion graph contract, release train
compatibility, install artifacts, and authoring behavior for the 0.43 product
minor.

The 0.43 Manual uses graph 0.1 as the active project and graph model. Start
with [Graph 0.1 Current Contract](model/graph-01-current.md) for
`ProjectDocumentV01`, patch libraries, live help, graph fragments, Runtime
graph targets, and the rule that unsupported versions are rejected with
structured diagnostics. Then read [Subpatches](model/subpatches.md),
[Live Help](model/live-help.md), and
[Graph Fragments](model/graph-fragments.md).

For release train policy, read [Manual Versions](manual-versions.md) and
[Release Train And Install Artifacts](release-train-install.md). They explain
strict v0 version rejection, lockstep 0.43 product train artifacts, Runtime
standalone binaries, Studio desktop sidecars, install artifact tiers, and the
rule that public Manual pages name exact current versions rather than broad
ranges or fallback aliases.

Runtime session coverage includes
[Desktop Runtime Sessions](model/desktop-runtime-sessions.md) and
[Realtime Collaborative Editing](model/realtime-collaboration.md).
