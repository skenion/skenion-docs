# skenion Manual 0.44

The 0.44 Manual track covers the skenion graph contract, compatibility matrix,
install artifacts, and authoring behavior for the 0.44 product minor.

The 0.44 Manual uses graph 0.1 as the active project and graph model. Start
with [Graph 0.1 Current Contract](model/graph-01-current.md) for
`ProjectDocumentV01`, patch libraries, live help, graph fragments, Runtime
graph targets, and the rule that unsupported versions are rejected with
structured diagnostics. Then read [Subpatches](model/subpatches.md),
[Live Help](model/live-help.md), and
[Graph Fragments](model/graph-fragments.md).

For release and install policy, read [Manual Versions](manual-versions.md) and
[Compatibility Matrix And Install Artifacts](release-train-install.md). They
explain strict v0 wire-version rejection, 0.44 Contracts compatibility,
Runtime standalone binaries, Studio desktop sidecars, install artifact tiers,
and the rule that public Manual pages avoid broad version promises outside the
matrix's canonical Contracts range.

For Runtime session behavior, read
[Desktop Runtime Sessions](model/desktop-runtime-sessions.md) and
[Realtime Collaborative Editing](model/realtime-collaboration.md).
